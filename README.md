## Oque é conjectura de Collatz:
A Conjectura de Collatz (também chamada de problema 3n + 1) é um enigma matemático simples. Pegue qualquer número inteiro positivo: se for par, divida-o por 2; se for ímpar, multiplique por 3 e some 1. Repita o processo com o novo número. A conjectura afirma que, independentemente do número, você sempre chegará a 1.
